About

I've progressively taken on more architectural and leadership scope throughout my career - from owning frontend architecture at Oneview Healthcare and Linkly, to leading a technical team at Cohort Go, to now owning full platform architecture and running a cross-functional engineering group at Urban Rest.

My recent work has focused on full-stack platform builds on AWS (Lambda, ECS Fargate, RDS/PostgreSQL), React/Next.js frontends, React Native mobile apps, and standing up CI/CD pipelines and DevOps practices. I've led cross-functional teams of local and offshore engineers, run SCRUM end-to-end, and made the architecture and trade-off calls that come with being the most senior technical person in the room.

I've been building with AI tools in both production and personal projects, using Claude, Claude Code, and direct API integrations to ship features and automate development workflows. I care about writing clear code, shipping things that work, and not overcomplicating what doesn't need to be complicated.
